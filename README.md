# Python_x_Math

A repository where I explore mathematics through Python.

This repository documents my journey through the "Master Math by Coding in Python" course by Mike Cohen, where mathematical concepts are explored and implemented using Python. As I progress through the course, I add code examples, explanations, and exercises that help reinforce both programming and mathematical thinking.

---

# Purpose

The goal of this repository is to:

Learn mathematics through coding
Improve Python programming skills
Build intuition for mathematical concepts
Maintain a public record of my learning journey

This repository is continuously evolving as I work through the course and explore new topics.
